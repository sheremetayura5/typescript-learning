## Setup
- Install deps: `npm install`
- Run lesson 1 in dev (ts-node): `npm run dev:lesson-1` or `npm run dev`
- Build all lessons: `npm run build`
- Run compiled lesson 1: `npm run start:lesson-1` or `npm start`

## Notes
- Strict TypeScript enabled.
- Structure: `src/lesson-1`, future lessons in `src/lesson-2`, etc. Output mirrors in `dist/lesson-*`.
- `printUserInfo` accepts up to 4 args: `name`, `dob`, `phone`, `note` and prints a formatted summary.
